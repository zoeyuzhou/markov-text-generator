Markov Chain Text Generator

This is a simple implementation of Markov chain text generator. A Markov chain algorithm basically determines the next most probable suffix word for a given prefix.

Follow below steps to try it:

1. Upload a text file which the generator will learn and generate from.

2. Enter the very first word as "Prefix" you want to generate from. For example: You

3. Enter the "Number of Words" you want to generate. For example: 100

4. Click "Generate" and wait....
